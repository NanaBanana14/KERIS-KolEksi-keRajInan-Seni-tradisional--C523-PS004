This is the first item's accordion body. It is shown by default, until the collapse plugin adds the appropriate classes that we use to style each element. These classes control the overall appearance, as well as the showing and hiding via CSS transitions. You can modify any of this with custom CSS or overriding our default variables. It's also worth noting that just about any HTML can go within the .accordion-body, though the transition does limit overflow.

Cara Login
  1. klik Masuk yang berada di pojok kanan atas
  2. Masukkan Email yang telah terdaftar
  3. Masukkan Password
  4. Klik Masuk
Cara Register
  1. klik Masuk yang berada di pojok kanan atas
  2. klik Mendaftar
  3. Masukkan Email
  4. Masukkan password (minimal 6 karakter)
  5. Klik Daftar
Melihat Galeri Seni
  1. klik Menu yang ada di Navbar
  2. Klik Galeri Seni
  3. Untuk Melihat kategori seni >
  4. Klik dropdown Kategori yang ada dipojok kiri atas
  5. Untuk Melihat informasi lebih banyak >
  6. Klik Read more > Detail Galeri Seni
Review Galeri Seni
  1. Masuk kehalaman detail Galeri Seni
  2. Scroll kepaling bawah >
  3. Tulis Ulasan > jika belum login, lakukan login terlebih dahulu
  4. jika sudah login
  5. Tuliskan Nama > Tuliskan Ulasan
  6. Kirim
Melihat Event dan News
  1. klik Menu yang ada di Navbar
  2. Klik News
  3. Event > Read More
  4. Menampilkan Detail Event
  5. News > Read More >
  6. Pindah ke Halaman News
Melihat Kalender Budaya
  1. klik Menu yang ada di Navbar
  2. Kalender Budaya
  3. Klik pada tanggal yang memiliki kegiatan
  4. Detail informasi kegiatan seni
